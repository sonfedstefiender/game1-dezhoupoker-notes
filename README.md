# game1-dezhoupoker-notes

本仓库用于归档和发布多个独立的 HTML 页面，作为学习与整理过程中的笔记与示例集合。

## 项目简介

这是一个以 HTML 页面形式存放各类笔记、演示页面、小工具或示例代码的仓库。页面之间相互独立，不依赖特定后端服务，可离线浏览或直接通过 GitHub Pages 等静态托管服务查看。

仓库本身不针对任何具体网站或域名，所有页面仅供技术学习与参考使用。

## 目录说明

```
.
├── index.html          # 仓库主页（可选）
├── pages/              # 存放具体 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源（如 CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：主要页面目录，每个 HTML 文件独立存放，文件名应体现页面内容。
- `assets/`：存放页面共用的样式、脚本、字体或图片等资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立可运行文件，可直接在浏览器中打开。
- 页面可能包含外部资源引用（如 CDN 库），建议使用常见稳定 CDN 或自行托管。
- 归档内容会不定期更新，新增或修改页面后会在仓库中同步记录。
